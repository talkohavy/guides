# Git Alias

## - Alias 1: `git logs`

```bash
git config --global alias.logs "log --graph --oneline --decorate"
git logs
```

---

## - Alias 2: `git ck`

```bash
git config --global alias.ck "checkout"
git ck
```

---

## - Alias 3: `git pop`

```bash
git config --global alias.pop "stash pop"
git pop
```

---

## - Alias 4: `git apply`

```bash
git config --global alias.pop "stash apply"
git apply
```
