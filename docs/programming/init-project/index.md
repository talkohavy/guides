# Init Project

Commands to bootstrap linting and formatting in a new project.

---

## Install ESLint

```bash
pnpm add -D eslint@latest @eslint/js@latest typescript-eslint@latest eslint-plugin-import@latest eslint-plugin-perfectionist@latest eslint-plugin-react-compiler@latest eslint-plugin-react-hooks@latest globals@latest
```

---

## Install Prettier

```bash
pnpm add -D prettier@latest
```

---

## Install Biome

```bash
pnpm add -D @biomejs/biome@latest
```
